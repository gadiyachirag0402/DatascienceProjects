Project Descriptions
Project 1 — Student Performance Analysis
Analyzed key factors affecting student academic performance such as study time, failures, internet access, and gender. Performed EDA using Pandas and visualized results using Matplotlib and Seaborn including bar charts, histograms, and a correlation heatmap.

Project 2 — Sales Data Analysis
Performed end-to-end sales data analysis on a retail dataset. Identified top products, regional performance, monthly trends, and profit/loss by sub-category. Used GroupBy aggregations and multiple chart types including pie, bar, line, and scatter plots.

Project 3 — Titanic Survival Analysis + ML
Cleaned and analyzed the Titanic dataset to find survival patterns by class, gender, and age. Built a Logistic Regression classifier using Scikit-learn and evaluated it using accuracy score, confusion matrix, and classification report.

Project 4 — Probability & Statistics
Implemented core probability and statistics concepts using NumPy and SciPy. Covered normal distribution, descriptive statistics (mean, median, variance, std dev), coin toss simulation, Z-score outlier detection, and uniform/exponential/binomial distributions.

Project 5 — Linear Algebra using NumPy
Implemented linear algebra operations as per SPPU syllabus. Created a 3x3 matrix, computed its determinant, found its inverse, and solved a system of linear equations (2x+3y=10, 4x+5y=20) using NumPy. Also computed eigenvalues, eigenvectors, and visualized the graphical solution.
